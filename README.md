# vue-fauna-use-demo-data

## Prerequisites
* Node 14
* [Netlify Cli](https://www.netlify.com/products/cli/): `npm install netlify-cli -g`
* 
## Project setup
* Create a `.env` file in root

```
FAUNA_KEY=secret
FAUNA_DOMAIN=db.{region}.fauna.com
```

* Install packages
```
npm install
```

## Netlify Dev
```
netlify dev
```
